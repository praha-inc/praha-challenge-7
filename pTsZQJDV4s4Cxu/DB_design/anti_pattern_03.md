# 課題1
ジャンルが増えた際に、テーブルを追加しなければならない。  
たとえば、「評論」のジャンルが増えた場合、「評論」テーブルを作成し、type=評論を追加する必要がある。  
また「Manga」テーブルと「Novel」テーブルでIdが被った場合を考慮し、常にbelongs_to_idとtypeの2カラムで確認する必要がある。
